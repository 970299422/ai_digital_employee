# AI数字员工问答应用 - 项目概述

## 项目目标
基于PPT设计需求，创建一个功能完整的企业级AI数字员工问答应用，实现多部门智能助手的流式对话体验。

## 文件结构
```

├── resources/              # 资源文件夹
│   ├── bg.jpg             # 背景图片
│   ├── avatar-hr.png      # 人事部头像
│   ├── avatar-finance.png # 财务部头像
│   ├── avatar-market.png  # 市场部头像
│   └── avatar-tech.png    # 技术部头像
```

## 核心功能模块

### 1. 主页面 (index.html)
- **布局结构**：左侧导航 + 右侧对话
- **部门选择**：4个部门AI助手切换
- **对话界面**：消息显示、输入、发送
- **流式输出**：打字机效果显示AI回复

### 2. 交互功能
- 部门切换动画
- 消息发送与接收
- 打字机效果
- 滚动自动跟随
- 快速问题建议

### 3. 视觉效果
- Pixi.js粒子背景
- Anime.js文字动画
- 悬停交互效果
- 加载状态动画

### 4. 知识库内容
- 人事部：考勤制度、请假流程、薪资福利
- 财务部：报销流程、发票要求、预算申请
- 市场部：品牌规范、活动流程、广告投放
- 技术部：开发规范、部署流程、技术文档

## 技术栈
- **前端框架**：原生HTML/CSS/JS
- **样式框架**：Tailwind CSS
- **动画库**：Anime.js, Pixi.js
- **数据可视化**：ECharts.js
- **物理引擎**：Matter.js
- **字体**：Noto Sans SC

## 用户体验流程
1. 进入应用 → 默认显示人事部AI
2. 选择部门 → 切换AI助手和知识库
3. 输入问题 → 发送并显示在对话中
4. AI回复 → 流式输出显示答案
5. 继续对话 → 支持多轮问答

## 响应式设计
- **桌面端**：左侧导航固定，右侧对话自适应
- **平板端**：布局保持，适当缩小间距
- **移动端**：导航折叠，全屏对话体验

## 性能优化
- 图片懒加载
- 动画防抖
- 消息虚拟滚动
- 资源预加载
